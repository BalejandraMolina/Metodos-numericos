Biseccion
Interpolacion Lineal
Ingresa los puntos conocidos (uno por linea: x, y) y el valor x a interpolar. El programa detecta automaticamente entre que par de puntos cae x y aplica la formula lineal. Ejemplo: 1000, 0.084 2000, 0.635 4000, 4.945 8000, 39.364 16000, 314.733
Interpolacion de Lagrange
Ingresa los puntos conocidos (uno por linea: x,y) y el valor x a interpolar. Ejemplo: 0, 1.787 5, 1.519 10, 1.307 20, 1.002 30, 0.796
Graficador de funciones
Escribe hasta 3 funciones para graficar. Usa x como variable: x^2-2, x^3-x-2, sin(x)*2, etc.
